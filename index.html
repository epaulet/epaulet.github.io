About Me

My interest in making websites started at fourteen years old when I made my own website with just HTML and CSS.
After taking a few college CS courses and working in business operations I realized that software development would be a better career choice for me.

I have since built my own full-stack web application using Ruby on Rails and Backbone.js with a RESTful API and MVC architecture along with other JavaScript, Ruby, and SQL-based projects. I have experience in test driven development (TDD), pair programming, complex algorithms, and implementing various types of data structures (DFS/BFS trees, hashes, etc).
